# Smart Student Performance Tracker

## Project Description

This is a menu-driven C++ program to manage student performance.
It allows adding, viewing, updating, and deleting student records.
The program also provides class statistics, top 5 students,
AI-based performance predictions, and report generation.

# Features

* Add new student records
* Update marks, attendance.
* View all students and search by roll, name, or department
* Class statistics and top 5 students ranking
* AI performance prediction system
* Generate individual and class reports
* Save data to file
